Description: My real-time AR translator/interpreter for American Sign Language (ASL). By leveraging the power of datasets, Computer Vision (CV), Machine Learning (ML), and a Fully Connected Network model this application is able to interpret ASL in real-time; translating each gesture into textual letters displayed as AR overlays.

Note: This application has only tapped into models which use 1/6th of the total dataset! I'm currently limited hardware-wise in training a model that fully ultizies the entire ASL dataset.

Instructions: Please select an AI/ML model and toggle mirroring (if needed). Then click "Start Translation" to proceed.
